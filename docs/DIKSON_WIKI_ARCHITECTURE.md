# Dikson Wiki Architecture

Wiki pages are Markdown documents with YAML frontmatter.

Supported knowledge types:

- fact
- decision
- hypothesis
- task
- source

Planned layout:

- `wiki/projects/`
- `wiki/sources/`
- `wiki/decisions/`
- `wiki/agents/`
- `wiki/tasks/`

Each page must preserve provenance and status. Hypotheses and model-generated proposals must not be represented as confirmed facts.
