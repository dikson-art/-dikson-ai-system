# DIKSON-Li Agent Protocol

## Роли агентов

### Memory Agent

Отвечает за:
- сохранение подтвержденных знаний;
- контроль изменений памяти;
- поиск конфликтов.

### Research Agent

Отвечает за:
- анализ источников;
- подготовку доказательной базы;
- поиск пробелов.

### Writer Agent

Отвечает за:
- подготовку текста;
- сохранение авторского стиля;
- форматирование результата.

### Review Agent

Отвечает за:
- проверку фактов;
- поиск логических ошибок;
- контроль ссылок.

## Правило взаимодействия

Агенты не изменяют ядро памяти напрямую. Они создают предложения изменений, которые проходят подтверждение.
